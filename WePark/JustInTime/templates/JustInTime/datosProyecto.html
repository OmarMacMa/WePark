Inicio Cómo surge este proyecto

La cantidad de espacios ocupados del estacionamiento del Tec Campus Guadalajara varía en función de la hora, día y zona, además de que en este pueden ocurrir choques.

Nuestro proyecto surge como solución en base al análisis de la cantidad de espacios ocupados.

PlaceholderEntrada al estacionamiento TEC campus GDL

Nuestros objetivos

Placeholder

Crear un modelo simulando un estacionamiento para probar el funcionamiento de los sensores y que envíen los datos a una base de datos.

Placeholder

Tener una base de datos en la que se almacene la información recabada por los sensores, además de poder realizar el análisis de esos datos.

Placeholder

Crear una página web funcional para que los usuarios puedan ver esta información reflejada de forma clara y comprensible, de tal forma que sea de utilidad para ellos a la hora ed buscar lugar de estacionamiento

Placeholder

Finalmente, crear en forma de prototipo un servicio para que pueda ser capitalizable y útil para el cliente.

Alcance

Se espera que el proyecto sea adoptado por el campus como una inversión para la comodidad de todos los que forman la comunidad Tec y así pase de una maqueta a prototipo y finalmente a su uso cotidiano
